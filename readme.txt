== Description ==

This plugin enables the Golang brush for Syntax Highlighter Evolved. You can use [go] or [golang] for posting your code.
View examples of this plugin working at http://luisivan.net/2011/06/01/golang-for-syntaxhighlighter/

== Installation ==

Simply install it as a normal plugin. Very easy
